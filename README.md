# Recall — Memory Assistant (Chrome Extension)
A sleek, minimal Chrome Extension to save notes, reminders, and bookmarks directly in your browser. Built with vanilla HTML, CSS, and JavaScript. Data is stored securely using the browser's native localStorage.

## 🚀Features
- 📝 Notes: Quickly jot down thoughts and ideas.
- 🔔 Reminders: Set date and time reminders with browser notifications.
- 🔖 Bookmarks: Save important URLs with context.
- 🏷️ Tags & Filters: Organize items with tags and filter by type.
- 🔍 Search: Instantly search through your saved items.
- 🌚 Dark Mode: Beautiful dark interface optimized for readability.

  
## 📦Installation
Install it locally:

1. Clone this repository or download the ZIP and extract it.
2. Open Google Chrome and go to `chrome://extensions/.`
3. Turn on **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** (top-left).
5. Select the `recall-extension` folder.
6. Pin the extension to your toolbar and click it to open!
